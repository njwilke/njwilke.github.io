Brass Monkey Fitness

A C# gym software application that allows users to book sessions, write trainer reviews, view trainer ratings, and purchase gym memberships. There is also an admin portal where a manager can add, edit, and delete trainers, sessions, and customer bookings. In addition to this, there is a reports option that shows sessions per customer, revenue breakdown by month and year, and average trainer ratings.

A full-stack project that allows customers to select a vending machine, select products to purchase, and then check out using a POS interface. There is also an admin side that allows the user to view sales reports, edit inventory, and add vending machines/products.
